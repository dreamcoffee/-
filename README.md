# 음식점 리뷰 분석
간단한 크롤러 만들기부터 시작하여 음식점 리뷰 분석까지하는 프로젝트

 # 목표
 음식점 리뷰를 크롤링을 통해 데이터를 모으고 분석하여 긍정/부정 결과를 나타내는 프로그램을 만들기
 -> 감성분석 & 긍정/부정 키워드 분석을 해야함 [공부 해야함]
      
# 진행상황
- 임시로 리뷰 데이터를 가져올 음식점을 정하고 해당 음식점의 리뷰 데이터를 가져옴 [요기요]
- 데이터를 txt파일로 만듦
